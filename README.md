O objetivo do projeto é ajudar desenvolvedores que utilizam IA a manter mais organização, contexto e segurança durante a criação de sistemas.

A proposta envolve conceitos como:

- Controle de escopo
- Organização de contexto

- Registro de decisões
- Checklist de segurança
- Revisão de código
- Apoio em testes
- Documentação do processo de desenvolvimento

Esse projeto surgiu a partir da minha própria experiência utilizando IA para construir sistemas e percebendo os riscos de depender dela sem um fluxo bem definido.
